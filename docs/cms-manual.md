# CMS 操作マニュアル

## ログイン

ブラウザで `http://cms.yourdomain.com`（または開発環境では `http://localhost:5173`）にアクセスします。

- **メールアドレス**: `admin@example.com`
- **パスワード**: `Admin1234!`

> ⚠️ 初回ログイン後は必ずパスワードを変更してください。

---

## ダッシュボード

ログイン後、ダッシュボードが表示されます。  
現在の記事数・カテゴリ数・ユーザー数の概要と、クイックアクションボタンが確認できます。

---

## 記事管理

### 記事一覧

サイドバーの「📝 記事管理」をクリックすると一覧が表示されます。

- **検索**: キーワードで記事タイトル・抜粋を絞り込めます
- **カテゴリフィルター**: ドロップダウンでカテゴリを絞り込めます
- **ステータスフィルター**: 「公開」「下書き」で絞り込めます
- **編集**: 各行の「編集」をクリックして編集画面へ
- **削除**: 各行の「削除」をクリックすると確認ダイアログが表示されます

### 通常記事・ブログ記事の作成

1. 「＋ 新規作成」ボタンをクリック
2. **タイトル**を入力（スラッグは自動生成されます）
3. **カテゴリ**を選択
4. **ステータス**を設定（「下書き」または「公開」）
5. **アイキャッチ画像 URL**を入力（任意）
6. **抜粋**を入力（一覧ページに表示される概要、任意）
7. **本文**をリッチテキストエディタで入力
   - **B** / **I**: 太字 / 斜体
   - **H2** / **H3**: 見出し
   - **•** / **1.**: 箇条書き / 番号リスト
   - **❝**: 引用ブロック
   - **</>**: コードブロック
8. 「作成する」または「下書き保存」をクリック

### YouTube記事の作成

1. 一覧画面の「▶ YouTube記事」ボタンをクリック
2. **YouTube URL または 動画ID**を入力して「動画情報を取得」をクリック
   - 対応形式: `https://www.youtube.com/watch?v=VIDEO_ID`、`https://youtu.be/VIDEO_ID`、`VIDEO_ID` のみ
3. 取得成功するとサムネイルとタイトルが自動入力されます
4. タイトル・スラッグ・ステータスを確認・編集
5. **説明文**を入力（任意）
6. 「作成する」をクリック

## カテゴリ管理

サイドバーの「🗂️ カテゴリ管理」をクリックします。

### カテゴリの追加

「＋ カテゴリを追加」ボタンをクリックしてモーダルを開きます。

| 項目 | 説明 |
|------|------|
| カテゴリ名 | 表示名（例：「旅行記」） |
| スラッグ | URL用識別子、半角英数字とハイフン（例：`travel`） |
| タイプ | `通常` / `ブログ` / `YouTube` / `カスタム` |
| 表示順 | 数字が小さいほど先に表示されます |

### カテゴリの編集・削除

一覧の「編集」または「削除」ボタンをクリックします。  
> ⚠️ カテゴリを削除すると、そのカテゴリに紐づいた記事はカテゴリなし（`NULL`）になります。

---

## ユーザー管理

サイドバーの「👤 ユーザー管理」をクリックします。

### ロールについて

| ロール | 権限 |
|--------|------|
| **管理者 (admin)** | すべての操作が可能 |
| **編集者 (editor)** | 記事の作成・編集・削除が可能 |

### ユーザーの追加

「＋ ユーザーを追加」ボタンをクリックします。  
名前・メールアドレス・パスワード・ロールを入力して保存します。

### ユーザーの編集

「編集」をクリックしてモーダルを開きます。  
パスワードは変更する場合のみ入力してください（空欄の場合は変更されません）。

---

## ログアウト

画面右上の「ログアウト」をクリックするとセッションが終了します。

---

## よくある質問

**Q: 記事を公開後に非公開に戻せますか？**  
A: はい。記事の編集画面でステータスを「下書き」に変更して保存すると非公開になります。

**Q: 画像をアップロードするには？**  
A: 現在はアイキャッチ画像URLを直接入力する方式です。画像は外部のストレージ（Amazon S3など）にアップロードし、URLを入力してください。

**Q: YouTube動画情報の取得に失敗します**  
A: インターネット接続とYouTube URLの形式を確認してください。動画IDのみ（11文字）でも入力できます。

**Q: スラッグを変更すると既存のURLが壊れますか？**  
A: はい。スラッグはURLに使用されるため、公開済み記事のスラッグ変更は注意が必要です。変更する場合はリダイレクト設定を検討してください。
